Portafolio

Un portafolio es una colección organizada de trabajos, proyectos, documentos o muestras que un individuo o una organización selecciona para demostrar sus habilidades, logros, experiencias y conocimientos en un área específica.

Planificacion

La planificación de un proyecto es el proceso de definir y organizar las tareas y recursos necesarios para alcanzar los objetivos del proyecto. Es una fase fundamental para el éxito del proyecto.

Desarrollo de proyecto

La planificación de un proyecto es el proceso de definir y organizar las tareas y recursos necesarios para alcanzar los objetivos del proyecto. Es una fase fundamental para el éxito del proyecto.

Finalizacion de proyecto

La finalización de un proyecto, también conocida como cierre de proyecto, es la etapa final de un proyecto. Consiste en formalizar el cumplimiento de los objetivos y actividades planificadas.
